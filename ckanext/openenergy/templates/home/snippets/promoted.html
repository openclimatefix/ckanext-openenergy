The Open Energy Data Portal provides an entry point to the world of energy data. It allows you to search for energy data providers and datasets and includes where available the metadata to describe the structure of the datasets. It fully supports the Data Catalog Vocabulary and also ensures all datasets can be found by e.g. the Google Dataset Search.

It forms part of the technical work of the consortium of Icebreaker One, Passiv Systems and Open Climate Fix for the Modernising Energy Data Access competition. All code is open-source and can be found on GitHub.

This portal will grow over time to reference more and more energy datasets. If you have datasets to link, please suggest them here.
